# 🏃 Motion Analysis - 물리치료 동작평가 AI 시스템

물리치료사 5년 임상 경험을 바탕으로 개발한
MediaPipe 기반 실시간 동작분석 시스템

## 📌 프로젝트 배경

5년간 물리치료사로 일하면서 스쿼트 동작평가를 직접 수행했다.
단순한 각도 기준보다 개인별 패턴 변화와 좌우 비대칭이
더 임상적으로 의미있다는 걸 경험했고, 이를 AI로 구현했다.

## ✅ 구현 기능

### 현재 (v1)
- 실시간 관절각도 계산 (코사인 법칙)
- 정면/측면 자동 감지 (어깨 x좌표 차이)
- 정면 모드: 좌우 무릎 비대칭 감지 (15도 이상 경고)
- 측면 모드: 무릎 굴곡각도 + 체간 앞쏠림 감지

### 예정 (v2)
- 스쿼트 동작 단계 자동 인식
- 최저점 무릎 각도 자동 기록
- 반복 동작 각도 변화 추적
- 피로 누적 패턴 감지

## 🗂️ 프로젝트 구조

scripts/
  pose_detection.py    # MediaPipe 관절 추출 기초
  angle_calculator.py  # 관절각도 계산
  squat_analyzer.py    # 정면/측면 통합 분석

## 🛠️ 기술 스택

- Python 3.10
- MediaPipe 0.10.9
- OpenCV
- NumPy
